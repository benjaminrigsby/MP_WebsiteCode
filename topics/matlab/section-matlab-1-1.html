MATLAB

Section 1.1 - History and Intro

Summary (TL;DR)

  • MATLAB is made for scientists and engineers.
  • MATLAB can do lots of things useful for mechanical engineers.
  • A software licence is required to use MATLAB, however options are available.
  • There are free alternatives to MATLAB that mimic MATLAB.

What is MATLAB?

MATLAB is a high-level interpreted numerical analysis programming language built for scientists and engineers in academia and industry. MATLAB is relatively user friendly and provides many out of the box tools and resources for math, controls, data analysis, data manipulation, data acquisition, and more (See full list here).

This software has built in functionalities like plotting and statistical analysis that many other popular programming languages do not, which makes it attractive to engineers who are more focused on the problem rather than the computer science happening behind the scenes. For example, MATLAB ofers a "mean" function to compute the average of a set of data. The user can simply use this function without specifically adding and dividing individual numbers.

What is MATLAB Practically Good For?

  • Data manipulation
    • Math operations, statistics, linear algebra, vector and matrix manipulation, optimization, and more...
  • Data Presentation
    • Plotting/graphing (2D/3D), animation (2D/3D), graphical user interface (GUI) application, remote deployment, and more
  • Hardware Communication and Control
    • Sensor reading, actuator control, signal processing, signal generation, and more
  • Media Processing
    • Computer/Machine Vision, audio recording/playback, frequency analysis, and more
  • And much much more...

While MATLAB is capable of all of these and more, this topic will only cover the essential utilitis, while presenting practical examples. These essentials can be further used to figure out the more advanced utlities.

How Can I Get MATLAB Onto My Computer?

MATLAB is a is a proprietary language held by its parent company MathWorks, which means you, your school, or your company needs to pay for a license to use it.

As of 2017, an standard single commercial licence costs $2650, while an educational licence to students costs $625. However, there are alternatives that mimic MATLAB. (See Section Below) MATLAB is highly modular, meaning that various and specialized toolboxes that can be added to the core software. While some toolboxes come standard with the MATLAB package, many need to be purchased, ranging from $500 to $1000. A couple examples are the Aerospace Toolbox for advanced aerospace analysis and development and the Instrument Control Toolbox to control and communicate with test and measurment instruments.

MATLAB Alternatives for FREE

Not every engineer or engineering company is able to afford a software licence to use MATLAB. The list below shows only the most popular numerical analysis software alternatives to MATLAB. In most cases these alternatives offer identical commands and syntax/code structure. A full list of all alternatives can be found here, while a comparison between all of them can be found here.

  1. GNU Octave
  2. SciLab
  3. FreeMat

Note that a lot of functions and problem solving abilities that are available to MATLAB can be accomplished with other high-level programming languages such as Python programming language, however do usually use a different syntax/code while also needing some extra setup and understanding.
